Place your custom audio files here if you want to keep assets with the project.

Suggested filenames:
- ambient.wav (or .mp3/.m4a/.ogg)
- beep.wav (or .mp3/.m4a/.ogg)

How to use in app:
1) Launch the simulation.
2) Click ENGAGE SYSTEM.
3) Use the CUSTOM AMBIENT and CUSTOM BEEP file selectors in the top-right audio panel.
4) Pick files from this sounds folder.

Notes:
- Ambient loops continuously.
- Beep clips trigger periodically with the interface feed.
- Use VOLUME and M key to control loudness/mute.
